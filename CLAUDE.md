## Key Conventions

### Language
- The README is written in **Spanish**. Maintain Spanish for all user-facing content unless explicitly told otherwise.

### Formatting
- Uses **centered HTML alignment** (`<h1 align="center">`, `<p align="center">`) for headers and footers.
- Uses **shield.io badges** for contact links.
- Uses **Markdown tables** for project listings.
- Uses **bold** for technology names and key terms.
- Sections are separated by horizontal rules (`---`).

### Content Style
- Professional, concise, recruiter-oriented tone.
- Focus on production experience and real-world outcomes.
- Technologies are grouped by domain (Frontend, Mobile, Backend, Auth, Payments, Web3, Architecture).
